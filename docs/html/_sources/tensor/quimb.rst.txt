quimb: Quantum Circuit and Tensor Network Development Library
=============================================================

**Package:** https://github.com/jcmgray/quimb

**Documentation:** https://quimb.readthedocs.io/en/latest/

quimb is an easy but fast python library for quantum information and many-body calculations, including with tensor networks. The code is hosted on github, do please submit any issues or pull requests there. It is also thoroughly unit-tested and the tests might be the best place to look for detailed documentation.

**Reference:** Gray, Johnnie. "quimb: A python package for quantum information and many-body calculations." *Journal of Open Source Software* **3**, (2018): 819. doi:`10.21105/joss.00819 <https://joss.theoj.org/papers/10.21105/joss.00819>`_
